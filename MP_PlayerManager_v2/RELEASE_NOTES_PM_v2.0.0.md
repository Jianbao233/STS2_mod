## MP_PlayerManager v2.0.0 — 更新说明（相对 v1）

### 新功能

- **复制玩家**：支持复制任意现有玩家（含装备、遗物、药水、地图进度），非仅全新初始状态。
- **Mod 角色支持**：自动扫描游戏 `mods` 目录下的 `player_template.json`，读取 Mod 角色初始属性与预设卡组。
- **Steam 头像与个人主页**：夺舍页面显示玩家头像与 Steam 个人主页链接。
- **中文 / English 界面**：随系统语言切换；左下角可手动切换语言。
- **内置角色初始血量**：与游戏数据对齐 — Ironclad 80、Silent 70、Defect 75、Necrobinder 66、Regent 75。
- **备份管理增强**：「全部备份」按 Steam 用户与存档 profile 分组浏览；支持恢复、跳转存档、删除备份；列表区域高度优化。

### Bug 修复（相对 v1）

- **存档写入格式**：改为与游戏一致的 CRLF 明文 JSON，避免 v1 曾出现的压缩写入导致读档损坏。
- **非房主昵称重复**：修复夺舍列表中 Steam 昵称与角色名显示重复的问题。
- **复制玩家数据**：药水列表不再写成 `null`；金币不再被错误固定为 100。
- **地图进度**：新增玩家后正确向 `map_point_history` 注入 `player_stats`，减少读档/同步异常。

### 存档兼容性

- v2 面向杀戮尖塔 2 原生存档（`schema_version=14`，CRLF 明文 JSON）。
- **若曾用 v1 工具处理过存档，建议在 v2 中操作前自行备份**；极端情况下可能出现与当前工具校验不一致的提示。

### 安装说明

解压后双击 `MP_PlayerManager-v2.0.0.exe` 运行即可，**无需**放入游戏目录。
