# 《杀戮尖塔》角色梗与游戏梗合集

> 本文档汇总自 Reddit、Steam 社区、B 站、贴吧等玩家社群的经典梗、调侃与黑话。部分内容为玩家戏谑，不代表官方设定。

---

## 一、角色相关梗

### 1. 铁甲战士（Ironclad）

| 梗名 | 说明 |
|------|------|
| **牌烧光我打什么？** | 腐化（Corruption）会让技能牌变为 0 费且使用后消耗，新手常误以为「牌烧光就没法打了」。实际上腐化配合黑暗之拥（抽牌）和无惧疼痛（护甲）可形成持续运转，是战士最强组合之一。 |
| **腐化 = 轮椅** | 腐化常被誉为「轮椅级」能力牌，一旦成型几乎碾压大部分战斗。 |
| **黑化无惧** | 腐化 + 黑暗之拥 + 无惧疼痛的经典组合，简称「黑化无惧」。 |
| **树枝腐化** | 枯木树枝（Dead Branch）配合腐化：每消耗一张牌就随机获得一张牌，可形成「无限流」般的牌海战术。 |
| **恶魔形态** | 每回合加力量但费用高、见效慢，常被调侃为「前期等死，后期无敌」。 |
| **卖灵魂的战士** | 因与恶魔交易获得力量，玩家常用「卖灵魂」形容铁甲战士的设定。 |

---

### 2. 静默猎手（Silent）

| 梗名 | 说明 |
|------|------|
| **毒伤永存** | 毒伤会持续到战斗结束，叠加后伤害非常可观，常被形容为「毒到就赢」。 |
| **小刀/飞刀流** | 以刀刃类牌为核心，配合精准、苦无等遗物，强调快速多段攻击。 |
| **无限流猎手** | 静默可依靠弃牌、过牌与少量关键牌达成无限循环，是无限流构建的代表角色之一。 |
| **精准+** | 精准是飞刀流的核心增益，玩家常戏称「有精准就拿，没精准就难受」。 |

---

### 3. 故障机器人（Defect）

| 梗名 | 说明 |
|------|------|
| **能量球管理大师** | 需要同时管理闪电、寒冰、暗影、等离子等多种能量球，操作复杂，常被调侃为「脑子不够用」。 |
| **卡死流** | 能量球过多或牌序不佳时，可能导致一回合打不出有效操作，俗称「卡死」。 |
| **暗影球流** | 暗影球需蓄力才能打出高伤，成型慢但爆发强，常被形容为「憋大招」。 |
| **Creative AI + 树枝** | 创意 AI 每回合生成随机能力牌，配合枯木树枝可形成持续随机出牌的「赌狗」玩法。 |
| **取档流** | 因随机性和操作空间大，玩家常通过 SL（存读档）反复尝试最优解，机器人尤甚。 |

---

### 4. 观者（Watcher）

| 梗名 | 说明 |
|------|------|
| **红蓝 yyds** | 愤怒（红）与平静（蓝）姿态切换是观者核心机制，被玩家奉为「永远的神」。 |
| **姿态舞** | Stance Dance，在愤怒与平静间来回切换以获取能量与伤害加成，是观者最主流玩法。 |
| **无限姿态** | 观者最容易达成无限流：小牌组 + Rushdown + 1 费愤怒/平静入口，可无限循环打牌。 |
| **亵渎** | Blasphemy 可瞬间进入神圣姿态获得三倍伤害，但下回合开始时直接死亡。玩家常戏称「用命换伤害」「赢了就是神，输了就是菜」。 |
| **点穴流** | 以 Pressure Points 为核心，不依赖姿态，但怕多目标和 artifact，常被调侃为「邪教流派」。 |
| **盲眼大师** | 观者设定为盲眼修行者，玩家常用「盲打」形容其高难度操作。 |

### 5. 摄政官 & 死灵法师（STS2 新角色）

## 二、游戏机制与遗物梗

### 1. 遗物相关

| 梗名 | 说明 |
|------|------|
| **树枝** | 枯木树枝（Dead Branch），消耗牌时随机获得一张牌。与腐化、创意 AI 等配合极强，但会破坏精简牌组策略。 |
| **勺子 vs 消耗战** | 某遗物与消耗机制冲突，不适合「烧牌」型战士，玩家总结为「勺子别配消耗战」。 |
| **顶级组合翻车** | 枯木树枝 + 异蛇之眼 + 符文金字塔有时会导致手牌全是高费或无用牌，被戏称「顶级组合，顶级翻车」。 |
| **谁需要遗物？** | 成就「Who Needs Relics?」要求只带一个遗物通关，是对「遗物越多越好」观念的调侃。 |

---

### 2. Boss 与敌人梗

| 梗名 | 说明 |
|------|------|
| **时间吞噬者** | Time Eater 每回合限制打出 12 张牌，超过即结束回合。对无限流、过牌流、小刀流等极不友好，玩家常称「时间吞噬者：无限流杀手」。 |
| **三萝莉 / 幼儿园** | 玩家社区对某些 Boss 或精英怪的戏称（如三位一体的 Boss 或低层精英）。 |
| **腐化心脏** | 最终 Boss，每回合最多承受 300 伤害，需要多回合输出或特殊对策，常被调侃「打心是另一款游戏」。 |
| **大脑袋** | 游戏中存在「 literally just a giant head」的敌人，成为社区 meme 素材。 |

---

### 3. 卡牌与流派梗

| 梗名 | 说明 |
|------|------|
| **极简流（Minimalist）** | 5 张牌以内通关的成就，需要大量删牌和运气，常被调侃「删牌删到破产」。 |
| **meme run** | 指不追求强度、以娱乐为主的局，如全拿某类废牌、故意不删牌等「整活」玩法。 |
| **Tier List 之争** | 社区对卡牌、遗物强度排名争论不休，不同流派、不同角色结论往往不同。 |
| **稀有度 ≠ 强度** | 常见共识：蓝卡不一定强，白卡不一定弱，关键看与当前牌组的契合度。 |

## 三、社区文化梗

### 1. 通用调侃

| 梗名 | 说明 |
|------|------|
| **SL 大法** | Save & Load，通过存档读档规避不利事件或重抽牌序，被戏称「时间回溯术」。 |
| **RNG 决定一切** | 强调随机性的影响，运气好时「随便打都赢」，运气差时「神仙也救不了」。 |
| **血是资源** | 在部分流派中，血量可用来换输出或关键遗物，玩家总结为「该卖血就卖血」。 |
| **删牌比买牌重要** | 新手易堆牌，老手强调删掉初始 Strike/Defend，此句常被用作「删牌优先」的口号。 |
| **Ascension 20** | 最高难度档位，玩家常以 A20 通关心碎为「毕业」标准。 |
| **碎心** | 击败腐化心脏，代表完整通关。 |

---

### 2. Reddit / Steam 社区梗

| 梗名 | 说明 |
|------|------|
| **SPIRIT POOP** | Reddit 上用于搞笑内容的标签之一。 |
| **Centurion and Mystic** | 某些精英怪组合被认为特别难缠，成为「好局翻车」的 meme 素材。 |
| **Take the hit** | 与护甲/格挡相关的社区梗，常出现在讨论「该不该硬吃伤害」时。 |
| **deck churn** | 形容牌组快速循环、大量过牌的打法，与 Dead Branch、Corruption 等配合时尤为明显。 |

### 3. 中文社区梗（B 站、贴吧等）

| 梗名 | 说明 |
|------|------|

| **拳打三萝莉，脚踢幼儿园** | 对轻松击败某些 Boss/精英的夸张形容。 |

| **红蓝 yyds** | 观者愤怒/平静姿态的流行说法。 |

## 四、梗来源与延伸

- **Reddit**：r/slaythespire 是英文区主要讨论区，meme、tier list、bug 反馈和创意内容集中于此。
